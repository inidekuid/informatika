Logika & Pemrograman Dasar

Dari Gerbang Logika ke Kode Pertamamu

Bahasa Komputer: 1 dan 0

Pernahkah kamu berpikir bagaimana komputer bisa melakukan hal-hal canggih? Rahasianya ada pada konsep yang sangat sederhana: Logika Boolean.

Komputer tidak mengerti bahasa manusia. Ia hanya mengerti dua kondisi, seperti saklar lampu: NYALA atau MATI. Dalam dunia komputer, kondisi ini diwakili oleh angka biner:

  • 1 atau TRUE (Benar / Ada Arus Listrik)
  • 0 atau FALSE (Salah / Tidak Ada Arus Listrik)

Gerbang Logika adalah sirkuit elektronik dasar yang menjadi "otak" komputer. Ia menerima satu atau lebih input (berupa 1 atau 0) dan menghasilkan satu output (juga 1 atau 0). Mari kita bedah cara kerjanya.

Gerbang AND: Si Perfeksionis

Gerbang AND hanya akan menghasilkan output TRUE (1) jika SEMUA inputnya TRUE. Jika salah satu saja inputnya FALSE (0), maka outputnya akan FALSE.

Analogi:

Bayangkan dua saklar yang terhubung seri (berderet) untuk menyalakan satu lampu. Agar lampu menyala, kedua saklar (Saklar A DAN Saklar B) harus dalam posisi ON.

Penjelasan Contoh:

Di samping, "Input A" dan "Input B" adalah saklarmu. Lampu di "Output" akan menyala (menjadi 1) hanya jika kedua saklar kamu geser ke posisi ON (biru).

Coba Sendiri!

Output

A AND B = 0

Gerbang OR: Si Fleksibel

Gerbang OR akan menghasilkan output TRUE (1) jika SALAH SATU atau lebih inputnya TRUE. Ia hanya akan FALSE jika semua inputnya FALSE.

Analogi:

Dua saklar yang terhubung paralel (bercabang). Kamu cukup menyalakan Saklar A ATAU Saklar B untuk membuat lampu menyala.

Penjelasan Contoh:

Coba geser salah satu saklar ke posisi ON. Lampu akan langsung menyala! Lampu hanya akan mati jika kedua saklar dalam posisi OFF.

Coba Sendiri!

Output

A OR B = 0

Gerbang NOT: Si Pemberontak

Gerbang NOT (atau Inverter) adalah yang paling unik. Ia hanya punya satu input dan tugasnya adalah MEMBALIKKAN nilainya. Jika inputnya 1, outputnya 0. Jika inputnya 0, outputnya 1.

Analogi:

Saklar "aneh". Jika kamu menekan ON, lampu justru mati. Jika kamu menekan OFF, lampu malah menyala.

Penjelasan Contoh:

Perhatikan, lampu sudah menyala (output 1) saat saklar OFF (input 0). Saat kamu menyalakan saklar, lampu justru akan mati.

Coba Sendiri!

Output

NOT A = 1

Peralatan Wajib Programmer: Operator

Setelah paham logika dasarnya, kita perlu alat untuk menggunakannya dalam kode. Itulah fungsi Operator. Anggap saja ini adalah "kata kerja" dalam bahasa pemrograman yang memungkinkan kita melakukan aksi, seperti menghitung dan membandingkan.

Operator Aritmatika & Perbandingan

Ini adalah operator paling umum. Aritmatika untuk berhitung (seperti kalkulator) dan perbandingan untuk membandingkan dua nilai yang hasilnya selalu `true` (benar) atau `false` (salah).

  • Penjumlahan: `+`
  • Pengurangan: `-`
  • Perkalian: `*`
  • Pembagian: `/`
  • Sisa Bagi (Modulus): `%` (Contoh: 10 % 3 hasilnya 1)
  • Sama Dengan: `==`
  • Tidak Sama Dengan: `!=`
  • Lebih Besar: `>`
  • Lebih Kecil: `<`
  • Lebih Besar / Sama Dengan: `>=`

Kalkulator Mini

?

Ubah angka atau operator, dan hasilnya akan langsung diperbarui. Ini menunjukkan bagaimana komputer terus-menerus mengevaluasi ekspresi.

Membuat Keputusan: `if-else`

Bagaimana komputer bisa merespons situasi yang berbeda? Dengan menggunakan kondisional. Ini adalah cara kita memberi instruksi pada komputer untuk mengambil "jalan" yang berbeda tergantung pada kondisi yang terpenuhi.

Analogi: "JIKA nilai ujian di atas 75, MAKA statusnya 'Lulus'. JIKA TIDAK, JIKA nilainya di atas 60, MAKA statusnya 'Remedial'. JIKA TIDAK (untuk semua kondisi lain), MAKA statusnya 'Tidak Lulus'."

Struktur Logika:

if (nilai >= 75) {
  hasil = "Lulus";
} else if (nilai >= 60) {
  hasil = "Remedial";
} else {
  hasil = "Tidak Lulus";
}

Penjelasan Kode:

Komputer akan mengecek kondisi dari atas ke bawah. Begitu satu kondisi `true` (misal: nilai 80, `80 >= 75` adalah `true`), ia akan menjalankan kode di dalamnya dan mengabaikan semua `else if` dan `else` di bawahnya.

Tugas Berulang: Looping

Looping atau perulangan adalah salah satu konsep paling kuat dalam pemrograman. Ini adalah cara menyuruh komputer melakukan tugas yang sama berkali-kali secara otomatis, tanpa kita perlu menulis kode yang sama berulang-ulang.

Loop `for`

Digunakan saat kita tahu pasti berapa kali perulangan akan dilakukan. Analogi: "Lakukan push-up sebanyak 10 kali."

Pseudocode:

ULANGI 5 KALI, mulai dari i = 1 CETAK "Menghitung: " + i SELESAI ULANGI

Contoh Kode (JavaScript):

for (let i = 1; i <= 5; i++) { console.log("Menghitung: " + i); }

Penjelasan: `let i = 1` (mulai dari 1), `i <= 5` (ulangi selama i kurang dari atau sama dengan 5), `i++` (tambahkan i dengan 1 setiap selesai perulangan).

Loop `while`

Digunakan selama sebuah kondisi masih bernilai `TRUE`, meskipun kita tidak tahu pasti berapa kali akan berulang. Analogi: "Teruslah aduk adonan SELAMA adonan masih cair."

Pseudocode:

SET hitungMundur = 3 SELAMA hitungMundur > 0 CETAK "Waktu tersisa... " + hitungMundur KURANGI hitungMundur dengan 1 SELESAI SELAMA CETAK "Boom!"

Contoh Kode (JavaScript):

let hitungMundur = 3; while (hitungMundur > 0) { console.log("Waktu tersisa... " + hitungMundur); hitungMundur--; } console.log("Boom!");

Penjelasan: Komputer akan terus mengecek `while (hitungMundur > 0)`. Jika `true`, kode di dalamnya dijalankan. Jika `false`, loop berhenti.

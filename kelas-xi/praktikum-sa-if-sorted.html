Strategi 1: Logika Manual (If-Else Bertingkat)

Strategi 2: Menggunakan Fitur Bawaan (List + Sort)

Contoh awal: "1. Masukkan semua nilai ke dalam sebuah daftar. 2. Gunakan perintah SORT untuk mengurutkan daftar dari besar ke kecil. 3. Tampilkan hasilnya."

üìö Bacaan Materi: Membedah Kode Strategi 2 (Klik untuk Buka)

Sebelum lanjut ke implementasi, mari kita pahami beberapa konsep kunci yang digunakan pada kode Strategi 2.

1. `dictionary`

Anggap saja dictionary seperti kamus kontak di ponsel. Kamu menyimpan data dalam format `key: value` (kunci: nilai). Dalam kasus ini, `key`-nya adalah nama siswa (contoh: "Ali") dan `value`-nya adalah nilainya (contoh: 85).

2. `sorted(data_siswa.items(), ...)`

Ini adalah inti dari pengurutan. Mari kita pecah:

  • `.items()`: Perintah ini mengubah `dictionary` menjadi sebuah daftar (`list`) yang berisi pasangan `(key, value)`. Jadi, `data_siswa.items()` akan menghasilkan `[('Ali', 85), ('Budi', 70), ('Citra', 95)]`.
  • `sorted()`: Ini adalah fungsi bawaan Python untuk mengurutkan. Fungsi ini akan mengambil daftar dari `.items()` dan mengurutkannya.

3. `key=lambda item: item[1]`

Ini adalah instruksi untuk fungsi `sorted()`. Ini memberi tahu `sorted()`: "Saat kamu membandingkan item, lihatlah elemen dengan index 1 (yaitu nilai) saja, bukan namanya (index 0)".

4. `reverse=True`

Secara default, `sorted()` mengurutkan dari kecil ke besar. Perintah `reverse=True` membalik urutannya menjadi dari besar ke kecil, sesuai yang kita butuhkan untuk membuat peringkat.

5. `enumerate(data_urut, start=1)`

Fungsi `enumerate()` secara otomatis memberikan nomor urut untuk setiap item dalam daftar. `start=1` berarti penomoran dimulai dari angka 1, bukan 0.

üöÄ Strategi 2: Kode List & Sort

Salin dan jalankan kode ini. Coba pahami setiap barisnya berdasarkan bacaan materi di atas.


# Membuat data dalam bentuk dictionary
data_siswa = {
    "Ali": 85,
    "Budi": 70,
    "Citra": 95
}

# Mengurutkan data berdasarkan nilai (elemen ke-1), dari terbesar (reverse=True)
data_urut = sorted(data_siswa.items(), key=lambda item: item[1], reverse=True)

print("\n--- Hasil Strategi 2 (List & Sort) ---")
# Menampilkan hasil yang sudah terurut
for i, (nama, nilai) in enumerate(data_urut, start=1):
    print(f"{i}. {nama} ({nilai})")
